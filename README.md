# Fizz Buzz-ing my way through languages

## The Rules

A program that:

- prints the numbers from `1` to `100`
- multiples of `3` print `"Fizz"`
- multiples of `5` print `"Buzz"`
- multiples of both `3` and `5` print `"FizzBuzz"`
- print the number itself otherwise
